# 🔗 Resources

A deliberately short list — only sources we actually use. If it's here, it earned its place.

## Official documentation

| Resource | Why it matters |
|---|---|
| [SharePoint developer docs](https://learn.microsoft.com/en-us/sharepoint/dev/) | The canonical reference for REST, CSOM and SPFx |
| [Microsoft Graph docs](https://learn.microsoft.com/en-us/graph/) | Everything beyond SharePoint's own API — profiles, mail, Teams, files |
| [SharePoint admin docs](https://learn.microsoft.com/en-us/sharepoint/) | Administration, service limits, governance |
| [Microsoft 365 roadmap](https://www.microsoft.com/en-us/microsoft-365/roadmap) | See what's coming before it hits your tenant |

## Tooling

| Resource | Why it matters |
|---|---|
| [PnP.PowerShell](https://pnp.github.io/powershell/) | *The* PowerShell module for SharePoint Online — used by most scripts in this vault |
| [CLI for Microsoft 365](https://pnp.github.io/cli-microsoft365/) | Cross-platform alternative, great for CI/CD pipelines |
| [Graph Explorer](https://developer.microsoft.com/en-us/graph/graph-explorer) | Try Graph calls without writing a line of code |
| [SP Editor](https://github.com/pnp/sp-editor) | Chrome DevTools extension for live SharePoint exploration and tinkering |

## Community

| Resource | Why it matters |
|---|---|
| [Microsoft 365 & Power Platform Community (PnP)](https://pnp.github.io/) | Samples, community calls, open-source projects |
| [sp-dev-docs issues](https://github.com/SharePoint/sp-dev-docs/issues) | Often the only place where obscure SPFx and API bugs are documented |
| [List Formatting samples](https://github.com/pnp/List-Formatting) | Copy-ready column and view formatting JSON |
| [SPFx web part samples](https://github.com/pnp/sp-dev-fx-webparts) | Hundreds of real SPFx implementations to learn from |
| [SharePoint look book](https://lookbook.microsoft.com/) | What modern SharePoint can look like when done well |
