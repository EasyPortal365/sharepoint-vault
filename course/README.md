# SharePoint Online – administrace od A do Z

> Kurzové materiály k praktickému školení pro IT Pro správce. Devět prakticky zaměřených kapitol popisujících klíčové informace o SharePoint Online z pohledu správce. Jediná sekce vaultu psaná **česky**.

Tohle jsou doprovodné materiály k živému kurzu, který vedu v OK System (3 dny, 8:30–16:00). Sdílím je otevřeně – ať už jste kurz absolvovali a chcete se k obsahu vracet, nebo jen hledáte souhrn k některému tématu.

## Co se v kurzu naučíte

- **Představení SharePointu** – pozicování produktu, klíčové scénáře, edice, rozdíly oproti SharePoint Serveru a přehled SharePoint Governance.
- **Tenanty z pohledu správce** – DEV/TEST/PROD tenanty, limity SharePoint Online, Admin Centrum, role na úrovni Microsoft 365 tenantu, typy skupin, doména a URL adresy webů.
- **Informační architektura** – terminologie, weby vs. podřízené weby, šablony a jejich synchronizace.
- **Správa podnikového obsahu** – seznamy, knihovny a jejich nastavení.
- **Oprávnění** – hierarchie, úrovně oprávnění, individuální oprávnění, zásady sdílení.
- **Vyhledávání** – indexace, vyhledávací dotazy, Modern Search, volitelně Hybrid Search.
- **Zálohování a obnova** – verzování, koše, zálohy webů/tenantů, Microsoft 365 Backup, řešení třetích stran.
- **Reporting** – Admin Center, přehledy využití, Microsoft Purview, zásady upozornění.
- **PowerShell** – SharePoint Online Management Shell, PnP PowerShell, příklady užití.
